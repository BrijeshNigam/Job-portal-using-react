# 🚀 BridgeLab: Your Ultimate Career Launchpad

Welcome to **BridgeLab**—not just another job board, but a comprehensive, all-in-one ecosystem designed to bridge the gap between ambitious talent and their dream careers. 

Built from the ground up with modern web technologies, BridgeLab empowers job seekers to seamlessly discover opportunities, gain deep insights into top companies, and rigorously prepare for every stage of the interview process. Whether you're sharpening your algorithms or mastering the HR round, BridgeLab has your back.

## ✨ Why BridgeLab? (Key Features)

- **💼 Dynamic Job Portal:** A beautifully crafted, intuitive interface to browse, filter, and apply for the most relevant industry openings.
- **📚 360° Prep Hub:** Stop bouncing between different websites. We bring the preparation directly to you:
  - **DSA Mastery:** Tackle complex data structures and algorithms with curated practice sets.
  - **Core Tech Fundamentals:** Brush up on Java, Python, DBMS, and OS essentials.
  - **Aptitude & Logic:** Sharpen your quantitative and logical reasoning skills.
  - **HR & Soft Skills:** Ace the behavioral rounds with confidence through proven tips and mock questions.
- **🏢 Deep Company Insights:** Don't just apply blindly. Dive into hiring trends, work culture, and what top companies are actively looking for.
- **🔐 Secure & Seamless Experience:** Robust user authentication ensures your data and progress remain completely private and secure.
- **📊 Personalized Dashboard:** Your command center. Track your applications, monitor your prep progress, and stay organized in one centralized hub.

## 🛠️ The Tech That Powers It

BridgeLab is engineered for speed, scalability, and a flawless user experience:

- **Frontend Magic:** React 19, Vite (for lightning-fast builds), React Router v7
- **Aesthetics:** Custom, pixel-perfect CSS paired with sleek Lucide React icons
- **State Management:** Seamless React Context API
- **Backend Architecture:** Express.js & MongoDB for a robust, scalable data foundation

## 🚀 Launch It Locally

Want to take BridgeLab for a spin? Getting started is a breeze.

### What You Need
Ensure you have [Node.js](https://nodejs.org/) installed.

### Quick Start Guide

1. **Grab the Code:**
   ```bash
   git clone <repository-url>
   cd Job-portal-using-react
   ```

2. **Install the Dependencies:**
   ```bash
   npm install
   ```

3. **Fire Up the Engine:**
   ```bash
   npm run dev
   ```

4. **Experience BridgeLab:** Open your favorite browser and head over to the terminal URL (usually `http://localhost:5173`).

## 📂 Architecture at a Glance

- `/src/pages/`: The core views driving the platform (Dashboard, Job Portal, Prep Hub, etc.)
- `/src/Components/`: The building blocks—reusable, beautifully styled UI components.
- `/src/Context/`: The brain behind the global state management.
- `/src/App.jsx`: The central nervous system routing everything together.
